{\LARGE \textbf{Writing a statistical paragraph --- Foundation}}\par\vspace{0.8em}
\noindent\textbf{Data set A: Favourite fruit} Apples 12, Bananas 8, Oranges 9, Grapes 5.\\[0.6em]
\textbf{1.} Write one statistical statement. Start with \emph{I notice that ...}\\[1.2em]
\textbf{2.} Write one evidence sentence using at least two numbers from data set A. Start with \emph{This is shown by ...}\\[1.2em]
\textbf{Data set B: After-school clubs} Football 14, Art 6, Music 10, Coding 4.\\[0.6em]
\textbf{3.} Write a 3-sentence statistical paragraph about data set B. Use: statement, evidence, context.\\[1.2em]
\textbf{Data set C: Reading time per night (students)} 0--19 min: 4, 20--39 min: 9, 40--59 min: 12, 60--79 min: 6.\\[0.6em]
\textbf{4.} Write one statistical statement and one evidence sentence about where most students are grouped.\\[1.2em]
\textbf{Data set D: Favourite lunch} Pizza 15, Wraps 11, Sushi 7, Salad 5.\\[0.6em]
\textbf{5.} Write one evidence sentence and one context sentence about the lunches.\\[1.2em]
\textbf{Data set E: Weekend activity (\%)} Sport 35, Gaming 25, Family 25, Other 15.\\[0.6em]
\textbf{6.} Write a 3-sentence statistical paragraph. Mention the largest category and one smaller category.\\[1.2em]
\textbf{7.} Put these parts in the best order for a statistical paragraph: context, evidence, statistical statement.\\[1.2em]
\textbf{8.} Rewrite this weak sentence so it becomes a statistical statement: \emph{Pizza is the best.}

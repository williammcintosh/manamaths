{\LARGE \textbf{Writing a statistical paragraph --- Excellence}}\par\vspace{0.8em}
\noindent\textbf{Data set A: Revision tools used} Class 1: Flashcards 9, Videos 15, Tests 12, Notes 6. Class 2: Flashcards 14, Videos 11, Tests 16, Notes 8.\\[0.6em]
\textbf{1.} Write a paragraph comparing the two classes. Include one similarity and one clear difference.\\[1.2em]
\textbf{Data set B: Training distance (km)} 0--2: 1, 2--4: 5, 4--6: 10, 6--8: 7, 8--10: 2.\\[0.6em]
\textbf{2.} Write a paragraph that summarises where most values are and what that suggests about the group.\\[1.2em]
\textbf{Data set C: Transport to school (\%)} Walk 34, Car 33, Bus 21, Bike 12.\\[0.6em]
\textbf{3.} Write a balanced paragraph that avoids overstating the gap between the two largest categories.\\[1.2em]
\textbf{Data set D: Cafe sales by day} Mon 21, Tue 23, Wed 27, Thu 26, Fri 19.\\[0.6em]
\textbf{4.} A student says, \emph{Wednesday was way better than every other day.} Rewrite this as a stronger statistical paragraph.\\[1.2em]
\textbf{Data set E: Goals across four games} Tui: G1 3, G2 1, G3 4, G4 2. Kaka: G1 2, G2 2, G3 3, G4 4.\\[0.6em]
\textbf{5.} Write a paragraph comparing the two teams, using at least two pieces of evidence.\\[1.2em]
\textbf{Data set F: Plant height (cm)} 5--9: 2, 10--14: 6, 15--19: 9, 20--24: 5, 25--29: 2.\\[0.6em]
\textbf{6.} Write a paragraph and include one possible contextual interpretation of the data.\\[1.2em]
\textbf{7.} Explain why a sentence like \emph{It is good} is not statistical writing.\\[1.2em]
\textbf{8.} Write a one-sentence checklist for a strong statistical paragraph.

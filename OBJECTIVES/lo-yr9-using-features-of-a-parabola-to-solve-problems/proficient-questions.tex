{\LARGE \textbf{Using features of a parabola to solve problems — Proficient}}\\[0.2em]
{\large Use features of each parabola to answer the problem.}
\begin{tasks}(3)
\task A tunnel is modelled by \mbox{$y=-(x-0)^2+16$}. What is the maximum height? What is the width at ground level?
\task A projectile follows \mbox{$y=-(x-2)^2+9$}. At which two \mbox{$x$}-values is it on the ground?
\task For \mbox{$y=(x+1)^2-4$}, write the interval of \mbox{$x$} for which the graph is below the \mbox{$x$}-axis.
\task For \mbox{$y=-(x-3)^2+1$}, write the interval of \mbox{$x$} for which the graph is above the \mbox{$x$}-axis.
\task For \mbox{$y=(x-0)^2-9$}, use symmetry to find the two points where \mbox{$y=-5$}.
\task For \mbox{$y=-(x+2)^2+4$}, use symmetry to find the second point if one point is \mbox{$(-1,3)$}.
\task A toy rocket follows \mbox{$y=-(x-4)^2+25$}. How high does it go? When does it land?
\task For \mbox{$y=(x-2)^2$}, explain why the graph only touches the \mbox{$x$}-axis once.
\task For \mbox{$y=\tfrac12(x-1)^2-2$}, estimate both \mbox{$x$}-intercepts to 1 decimal place.
\task For \mbox{$y=-\tfrac12(x+1)^2+3$}, estimate both \mbox{$x$}-intercepts to 1 decimal place.
\task For \mbox{$y=(x+4)^2-9$}, one root is \mbox{$x=-1$}. Use symmetry to find the other root.
\task For \mbox{$y=-(x-1)^2+8$}, for which \mbox{$x$}-values is the graph non-negative?
\end{tasks}

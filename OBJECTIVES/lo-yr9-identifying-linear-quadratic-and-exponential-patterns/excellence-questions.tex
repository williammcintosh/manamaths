{\LARGE \textbf{Identifying linear, quadratic, and exponential patterns — Excellence}}
\begin{tasks}(3)
\task For \mbox{$2,\ 7,\ 12,\ 17,\ 22$}, name the pattern type and describe the evidence.
\task For \mbox{$1,\ 4,\ 9,\ 16,\ 25$}, name the pattern type and describe the evidence.
\task For \mbox{$5,\ 15,\ 45,\ 135,\ 405$}, name the pattern type and describe the evidence.
\task Two students debate \mbox{$3,\ 8,\ 15,\ 24,\ 35$}. One says linear, one says quadratic. Who is correct? Explain.
\task A sequence has constant first differences. Could it be quadratic? Explain.
\task A sequence has constant second differences. Could it be exponential? Explain.
\task A sequence multiplies by \mbox{$0.5$} each term. Is it still exponential? Explain.
\task Classify and justify: \mbox{$81,\ 27,\ 9,\ 3,\ 1$}
\task Classify and justify: \mbox{$10,\ 6,\ 2,\ -2,\ -6$}
\task Classify and justify: \mbox{$6,\ 11,\ 18,\ 27,\ 38$}
\task Write a rule a teacher could use to identify a linear pattern from a list of terms.
\task Write a rule a teacher could use to identify an exponential pattern from a list of terms.
\task Pattern A has first differences \mbox{$6,\ 6,\ 6$}. Pattern B has ratios \mbox{$\times 2,\ \times 2,\ \times 2$}. Name both types.
\task A graph of a pattern curves upward more and more steeply. Which pattern type is most likely? Explain.
\task A graph of a pattern is a straight line. Which pattern type is most likely? Explain.
\task A graph of a pattern bends but does not grow by a constant ratio. Which type is most likely: quadratic or exponential? Explain.
\end{tasks}

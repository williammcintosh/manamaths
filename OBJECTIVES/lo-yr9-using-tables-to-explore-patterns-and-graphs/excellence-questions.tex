{\LARGE \textbf{Using tables to explore patterns and graphs — Excellence}}\\[0.2em]
Use table patterns to justify rules, compare graphs, and predict unknown values.

\begin{tasks}(3)
\task The table is $(0,4)$, $(1,7)$, $(2,10)$, $(3,13)$. Write the rule, then explain how the table shows the graph has constant gradient.

\task The table is $(1,12)$, $(2,9)$, $(3,6)$, $(4,3)$. Write the rule and predict where the graph crosses the $x$-axis.

\task The table is $(0,1)$, $(1,4)$, $(2,9)$, $(3,16)$. Explain why these points would not lie on one straight line.

\task {Complete the table for $y=3x-5$. Then state both intercepts.\\[0.3em]
\begin{tabular}{|c|c|c|c|c|}
\hline
$x$ & $0$ & $1$ & $2$ & $5$ \\\hline
$y$ &  &  &  &  \\\hline
\end{tabular}}

\task Complete the table for $y=18-2x$. Then decide whether the graph reaches $y=0$ before or after $x=10$.

\task A machine rule makes the table $(0,2)$, $(2,6)$, $(4,10)$, $(6,14)$. Write the rule in the form $y=mx+c$.

\task Two tables are shown. Table A: $(0,1)$, $(1,3)$, $(2,5)$. Table B: $(0,1)$, $(1,4)$, $(2,7)$. Which graph is steeper? Explain using the change in $y$.

\task A student says the table $(0,7)$, $(1,5)$, $(2,3)$, $(3,1)$ has rule $y=7-2x$. Check the rule using the table, then find the point for $x=5$.

\task The table is $(0,-1)$, $(1,2)$, $(2,5)$, $(3,8)$. Predict the point where $y=14$. Show the pattern you used.

\task The points in a table are $(0,10)$, $(5,8)$, $(10,6)$, $(15,4)$. Describe the graph and find a rule that fits these values.

\task A graph is made from the rule $y=\frac{3}{2}x+1$. Write a table for $x=0,2,4,6$, then explain why these values are useful for plotting.

\task The table is $(1,3)$, $(2,6)$, $(3,11)$, $(4,18)$. The first differences are not constant. What does that suggest about the graph?

\task One pattern uses $y=2x+1$. Another uses $y=2x+5$. Explain how the tables would show the graphs are parallel.

\task The table for a line includes $(2,9)$ and $(5,15)$. Find the gradient from the table, then write a rule for the line.
\end{tasks}

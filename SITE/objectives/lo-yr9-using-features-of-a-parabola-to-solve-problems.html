Proficient

12 web questions

Download PDF
  1. A tunnel is modelled by $y=-(x-0)^2+16$. What is the maximum height? What is the width at ground level?
  2. A projectile follows $y=-(x-2)^2+9$. At which two $x$-values is it on the ground?
  3. For $y=(x+1)^2-4$, write the interval of $x$ for which the graph is below the $x$-axis.
  4. For $y=-(x-3)^2+1$, write the interval of $x$ for which the graph is above the $x$-axis.
  5. For $y=(x-0)^2-9$, use symmetry to find the two points where $y=-5$.
  6. For $y=-(x+2)^2+4$, use symmetry to find the second point if one point is $(-1,3)$.
  7. A toy rocket follows $y=-(x-4)^2+25$. How high does it go? When does it land?
  8. For $y=(x-2)^2$, explain why the graph only touches the $x$-axis once.
  9. For $y=\tfrac12(x-1)^2-2$, estimate both $x$-intercepts to 1 decimal place.
  10. For $y=-\tfrac12(x+1)^2+3$, estimate both $x$-intercepts to 1 decimal place.
  11. For $y=(x+4)^2-9$, one root is $x=-1$. Use symmetry to find the other root.
  12. For $y=-(x-1)^2+8$, for which $x$-values is the graph non-negative?

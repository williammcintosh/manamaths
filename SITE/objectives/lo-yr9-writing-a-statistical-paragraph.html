  1. Write a 3-sentence paragraph comparing the two year levels. Include one similarity and one difference. \textbf{Data set B: Daily screen time (students)} 0--1 hr: 3, 1--2 hr: 7, 2--3 hr: 12, 3--4 hr: 8, 4--5 hr: 2. \textbf{2.} Write a 3-sentence paragraph about where most of the data sits. Use exact values. \textbf{Data set C: How students get news (\%)} Social media 45, Family 20, TV 15, Friends 20. \textbf{3.} Write a 3-sentence paragraph that mentions the largest category and one smaller category. \textbf{Data set D: Books borrowed by genre} Mystery 18, Fantasy 14, Comics 12, Science 10, Sport 6. \textbf{4.} Write a paragraph with at least two exact values from data set D. \textbf{Data set E: Homework completed} Class A: Mon 14, Tue 16, Wed 12, Thu 10. Class B: Mon 12, Tue 13, Wed 15, Thu 14. \textbf{5.} Write a 3-sentence comparison paragraph about the two classes. \textbf{Data set F: Bus trip times (minutes)} 0--9: 2, 10--19: 8, 20--29: 11, 30--39: 6, 40--49: 3. \textbf{6.} Write a paragraph and suggest what the data might mean for most students. \textbf{7.} Improve this evidence sentence so it is precise: \emph{Most people chose social media. It is bigger.} \textbf{8.} Explain why a good statistical paragraph needs exact values, not just words like \emph{bigger} or \emph{more}. \end{document}

  1. Write a paragraph comparing the two classes. Include one similarity and one clear difference. \textbf{Data set B: Training distance (km)} 0--2: 1, 2--4: 5, 4--6: 10, 6--8: 7, 8--10: 2. \textbf{2.} Write a paragraph that summarises where most values are and what that suggests about the group. \textbf{Data set C: Transport to school (\%)} Walk 34, Car 33, Bus 21, Bike 12. \textbf{3.} Write a balanced paragraph that avoids overstating the gap between the two largest categories. \textbf{Data set D: Cafe sales by day} Mon 21, Tue 23, Wed 27, Thu 26, Fri 19. \textbf{4.} A student says, \emph{Wednesday was way better than every other day.} Rewrite this as a stronger statistical paragraph. \textbf{Data set E: Goals across four games} Tui: G1 3, G2 1, G3 4, G4 2. Kaka: G1 2, G2 2, G3 3, G4 4. \textbf{5.} Write a paragraph comparing the two teams, using at least two pieces of evidence. \textbf{Data set F: Plant height (cm)} 5--9: 2, 10--14: 6, 15--19: 9, 20--24: 5, 25--29: 2. \textbf{6.} Write a paragraph and include one possible contextual interpretation of the data. \textbf{7.} Explain why a sentence like \emph{It is good} is not statistical writing. \textbf{8.} Write a one-sentence checklist for a strong statistical paragraph. \end{document}

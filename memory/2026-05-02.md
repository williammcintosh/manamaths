## LO11 (`lo-yr9-reading-scales`) — Final Layout

### Page layout
- **Single row per page**: all diagrams + questions fit on one page
- **Document class**: `extarticle 20pt A4 landscape`
- **Margins**: `margin=0.3cm,bottom=0.6cm`
- **Question font**: `\normalfont\bfseries\normalsize` (consistent 20pt bold across all pages)
- **Title system**: `\loTitle{Title}{scaffold_level}` with `\Huge\bfseries` colorbox, green leaf rule (`3pt` line)
- **Diagrams wrapped in**: `\resizebox{0.95\linewidth}{!}{...}` to fit horizontally

### TikZ macros defined inline in each file's preamble

| Macro | Args | Description |
|-------|------|-------------|
| `\Thermometer` | `{label}{value}` | Vertical thermometer -30 to 30, red fill to value/10+1, 5pt outline, 3pt ticks, 28pt labels |
| `\Ruler` | `{label}{value}` | 0-15 horizontal ruler, even labels only, red cross-line at value, 5pt stroke |
| `\DialScale` | `{label}{value}` | 0-30 round dial, angle = 90 - (value*12), 5pt circle, 4 labels only (0/10/20/30) |
| `\Speedometer` | `{label}{value}` | Semi-circle 0-100, angle = 180 - (value*1.8), km/h label, 5pt arc |
| `\GasDial` | `{label}{value}{dir}` | 0-9 small dial, dir=1 CW, dir=-1 CCW (mirror numbering) |
| `\GasMeter` | `{label}{d1}{d2}{d3}{d4}` | 4 alternating gas dials (CW/CCW/CW/CCW) in a row |
